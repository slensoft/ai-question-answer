# 提问方法论学习与实践平台

一个交互式的学习平台，帮助你掌握15种核心提问与思考方法论。

## 项目结构

```
public/
├── index.html                      # 主页面
├── README.md                       # 项目说明
├── 使用说明.md                     # 使用指南
├── 快速开始.md                     # 快速上手
├── 功能说明.md                     # 功能详解
├── 更新日志.md                     # 更新记录
├── css/
│   └── methodology-style.css       # 样式文件
└── js/
    ├── methodology-data.js         # 方法论数据
    └── methodology-app.js          # 应用逻辑
```

## 功能特点

### 1. 首页展示
- **方法论网格**：展示所有15种方法论，包含名称、类别、描述、难度和标签
- **搜索功能**：实时搜索方法论
- **分类筛选**：按类别快速筛选（思维模型、结构化提问、深度追问等）

### 2. 场景选择
- 工作场景：向领导汇报、项目复盘、团队会议等
- 学习场景：学习新知识、阅读理解、向专家请教等
- 创新场景：产品创新、头脑风暴、突破僵局等
- 决策场景：技术选型、风险评估、资源分配等

### 3. 智能决策树
- 通过回答几个问题，智能推荐最适合的方法论
- 交互式引导，降低选择难度

### 4. 动态问答系统
- 每个问题都有独立的输入框
- 问题和答案清晰对应
- 可以只回答部分问题
- 实时保存和查看

### 5. 学习记录
- 自动保存到浏览器 localStorage
- 显示最近10条练习记录
- 点击查看完整详情
- 可下载完整 JSON 格式的历史记录
- 可导出单条记录

## 包含的方法论

### 思维模型类
1. **5W2H 分析法** - 全面分析问题的7个维度
2. **黄金圈法则** - 从Why到How再到What
3. **第一性原理** - 从底层逻辑重新推理
4. **MECE 拆解法** - 相互独立、完全穷尽

### 结构化提问类
5. **SCQA 模型** - 情境-冲突-疑问-回答
6. **STAR 法** - 情境-任务-行动-结果
7. **PREP 法** - 观点-理由-案例-重申

### 深度追问类
8. **5 Why 法** - 连续追问找根因
9. **苏格拉底式提问** - 六类问题层层递进

### 决策分析类
10. **权衡矩阵** - 理性对比多个方案
11. **SPIN 提问法** - 引导式销售沟通

### 创新突破类
12. **逆向提问** - 从反面思考问题
13. **问题重构** - 换个角度重新定义

### 学习成长类
14. **费曼技巧** - 用简单语言检验理解
15. **双钻模型** - 发散-收敛的系统方法

## 使用方法

### 方式一：直接选择方法论
1. 在首页浏览所有方法论
2. 点击感兴趣的方法论卡片
3. 阅读详细介绍和示例
4. 填写实践表单
5. 保存学习记录

### 方式二：按场景选择
1. 在"按场景选择"区域选择你的场景
2. 选择具体需求
3. 系统推荐最适合的方法论
4. 开始实践

### 方式三：使用决策树
1. 在"智能决策树"区域开始
2. 回答几个简单问题
3. 系统智能推荐方法论
4. 开始实践

## 数据存储

所有实践记录保存在浏览器的 localStorage 中，包含：
- 时间戳
- 使用的方法论
- 问题描述
- 分析回答
- 反思总结

可以随时下载为 JSON 文件备份。

## 技术栈

- 纯 HTML/CSS/JavaScript
- 无需服务器，可直接在浏览器中运行
- 响应式设计，支持移动端

## 开发说明
