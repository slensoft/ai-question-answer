# 快速开始

本项目当前为 **Nuxt 4 + Vue 3**，主站与文档站分开运行。

## 环境要求

- Node.js: 建议 `>=20`
- npm: 建议 `>=10`

## 安装依赖

```bash
npm install
```

## 启动主站

```bash
npm run dev
```

默认地址：`http://localhost:3000`

## 启动文档站（VitePress）

```bash
npm run docs:dev
```

默认地址：`http://localhost:5173`

## 常用命令

```bash
# 主站构建
npm run build

# 文档构建
npm run docs:build

# 文档预览
npm run docs:preview
```

## 目录说明

```text
app/                      # Nuxt 应用源码
  pages/                  # 页面路由
  components/             # 组件
  api/                    # 前端业务层 API（localStorage 模拟）
  lib/                    # 方法论与工具数据
  types/                  # 类型定义
server/api/               # Nitro 服务端接口
docs/                     # VitePress 文档目录
  .vitepress/config.ts    # 文档站配置
```

## 核心入口

- 首页：`/`
- AI 引导：`/ai-guide`
- 方法论总览：`/methodology/all`
- 用户中心：`/user`
- 可视化：`/visualization`

## 常见问题

### 1. 启动时报 Node 兼容问题
请切换 Node 20+ 后重新安装依赖。

### 2. 文档站样式/页面异常
删除缓存后重启：

```bash
rm -rf docs/.vitepress/cache .nuxt
npm run docs:dev
```
