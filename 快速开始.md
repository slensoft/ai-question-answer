# 🚀 OpenRouter 快速开始

## 方式一：使用脚本（推荐）

### 完整测试（推荐）
```bash
./test-openrouter.sh
```

这个脚本会：
1. 检查 .env 配置
2. **真实调用 OpenRouter API 测试**（可选）
3. 启动开发服务器

### 仅测试 API
```bash
npm run test:openrouter
# 或
node test-openrouter-api.js
```

这个命令会：
- 真实调用 OpenRouter API
- 测试 3 个免费模型
- 显示详细的测试结果
- 不启动服务器

## 方式二：手动配置

### 1. 配置 API Key

```bash
# 复制示例文件
cp .env.example .env

# 编辑 .env 文件
# 将 OPENROUTER_API_KEY 改为你的真实 API Key
```

### 2. 测试 API（推荐）

```bash
npm run test:openrouter
```

### 3. 启动项目

```bash
npm run dev
```

### 4. 打开测试页面

浏览器访问：<http://localhost:3000/openrouter-test>

## 🎯 测试页面功能

### Web 测试页面
- ✅ 实时聊天测试
- ✅ 多模型选择
- ✅ 临时 API Key 输入
- ✅ 对话历史显示
- ✅ 错误提示

### 命令行 API 测试
```bash
npm run test:openrouter
```

- ✅ 真实调用 OpenRouter API
- ✅ 测试 3 个免费模型
- ✅ 显示响应内容

## ❓ 常见问题

### Q: 在哪里获取 API Key？
A: 访问 <https://openrouter.ai/> 注册并创建 API Key

### Q: 免费模型有限制吗？
A: 有速率限制，但足够开发测试使用

### Q: 如何切换模型？
A: 在测试页面的下拉菜单中选择

### Q: 中国地区能用吗？
A: 可以，推荐使用 Qwen 2.5 7B 模型
