# 全局导航栏实现说明

## 概述
已将导航栏抽象为全局组件 `GlobalNav`，并集成到所有页面的顶部。

## 实现内容

### 1. 创建全局导航组件
- **文件**: `src/components/GlobalNav.tsx`
- **功能**: 
  - 包含所有主要导航链接（首页、AI引导、按场景选择、智能决策树、所有方法论、用户中心）
  - 自动高亮当前页面
  - AI引导带有"NEW"标签和动画效果
  - 响应式设计，支持移动端

### 2. 集成到根布局
- **文件**: `src/app/layout.tsx`
- **修改**: 
  - 导入 `GlobalNav` 组件
  - 在 `<body>` 标签内、`{children}` 之前添加导航栏
  - 更新页面标题和描述为中文

### 3. 移除局部导航
从以下页面移除了 `MethodologyNav` 组件：
- `src/app/methodology/page.tsx`
- `src/app/methodology/all/page.tsx`
- `src/app/methodology/scenarios/page.tsx`
- `src/app/methodology/[slug]/page.tsx`
- `src/app/methodology/decision-tree/page.tsx`
- `src/app/user/page.tsx`

### 4. 样式更新
- **文件**: `src/app/globals.css`
- **新增样式**:
  - `.global-nav` - 固定在顶部的导航栏
  - 渐变紫色背景
  - 悬停和激活状态动画
  - 响应式布局（移动端自适应）
  - AI引导按钮的脉冲动画效果

## 导航栏特性

### 视觉效果
- 渐变紫色背景 (#667eea → #764ba2)
- 半透明白色按钮背景
- 悬停时上移动画
- 当前页面高亮显示
- AI引导按钮特殊渐变色和脉冲动画

### 响应式设计
- **桌面端**: 水平排列，左右分布
- **平板端**: 垂直堆叠，居中对齐
- **移动端**: 仅显示图标，隐藏文字标签

### 导航项
1. 🏠 首页 - 跳转到根路径
2. 🤖 AI 引导 (NEW) - 智能推荐功能
3. 🎯 按场景选择 - 场景分类浏览
4. 🌳 智能决策树 - 问答式引导
5. 📚 所有方法论 - 完整列表
6. 👤 用户中心 - 个人数据和记录

## 技术细节

### 固定定位
```css
position: sticky;
top: 0;
z-index: 1000;
```

### 路径匹配
使用 Next.js 的 `usePathname()` hook 自动检测当前页面并高亮对应导航项。

### 动画效果
- 悬停时向上移动 2px
- AI引导按钮的脉冲动画（2秒循环）
- 平滑的颜色和阴影过渡

## 使用说明

导航栏已自动集成到所有页面，无需在各个页面中单独引入。如需修改导航栏：

1. **修改导航项**: 编辑 `src/components/GlobalNav.tsx` 中的 `navItems` 数组
2. **修改样式**: 编辑 `src/app/globals.css` 中的 `.global-nav` 相关样式
3. **添加新页面**: 在 `navItems` 数组中添加新的导航项配置

## 注意事项

- 导航栏使用 `sticky` 定位，会随页面滚动保持在顶部
- 所有页面内容会自动添加顶部内边距，避免被导航栏遮挡
- 旧的 `MethodologyNav` 组件已不再使用，但保留在代码库中以防需要
